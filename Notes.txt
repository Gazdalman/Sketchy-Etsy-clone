Individual databases added for each of us. When we push changes to main will be when we also consolidate our databases manually to prevent merge conflicts.

The following will install all dependecies found in the Pipfile:
pipenv install
OR
pipenv install -r requirements.txt

Run seeds - from pipenv shell:
flask seed all

Set the date name format for alembic if you have to erase your migrations folder
file_template = %%(year)d%%(month).2d%%(day).2d_%%(hour).2d%%(minute).2d%%(second).2d_%%(slug)

Build from Dockerfile:
docker build - < Dockerfile
